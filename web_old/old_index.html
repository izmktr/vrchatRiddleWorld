VRChatワールド情報管理システム
VRChatの謎解きワールドやエスケープルームワールドの情報を収集・管理し、美しいインターフェースで閲覧できるWebアプリケーションです。
🏠
VRChatワールド一覧
収集したVRChatワールドの詳細情報を閲覧・検索・フィルタリングできます。サムネイル画像、作者情報、ワールドの説明、定員などの情報を確認でき、データの編集・削除も可能です。
📊
API エンドポイント
VRChatワールドデータのJSON APIです。開発者向けに生データへのアクセスを提供しています。外部アプリケーションとの連携に使用できます。
📈
統計情報
システムの統計情報や分析データを提供します。収集したワールド数、更新頻度、人気度分析などの情報を確認できます。
🔧
システム状態
システムの稼働状態とFirebaseデータベースの接続状況を確認できます。管理者向けのヘルスチェック機能です。
